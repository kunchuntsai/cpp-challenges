üéØ Problem:

Given an array of numbers, find the length of the longest increasing subsequence.

üß† Algorithm:

Use DP: dp[i] = length of LIS ending at i. For each j < i, if nums[j] < nums[i], dp[i] = max(dp[i], dp[j]+1).

üìã Step-by-Step Example

Example: nums = [10,9,2,5,3,7,101,18]

üîç How DP Works

  • Each dp[i] is the length of the LIS ending at i.
  • For each j < i, if nums[j] < nums[i], update dp[i].
